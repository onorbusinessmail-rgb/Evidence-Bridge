# Evidence-Bridge

PCからAndroidへファイルを爆速で転送する、Go言語製のツールです。

## 特徴
* **爆速転送:** ADBを活用した高速なファイル転送を実現。
* **簡単操作:** データ転送をスムーズに連携。
* **環境:** Windows PCからAndroid端末への転送に対応。

# Evidence-Bridge (English)

A blazing fast tool developed in Go for transferring files from PC to Android.

## Features
* **Blazing Fast Transfer:** High-speed file transfer leveraging ADB.
* **Easy Operation:** Designed for a seamless data transfer workflow.
* **Environment:** Supports transfer from Windows PC to Android devices.
